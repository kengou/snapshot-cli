Credentials
-----------
The CronJobs read the OS_* environment variables from the Secret "{{ .Values.openstack.secretName }}".
{{- if .Values.openstack.createSecret }}
The chart created this Secret from openstack.credentials.
{{- else }}
This Secret is NOT managed by the chart — make sure it exists in namespace
{{ .Release.Namespace }} with keys OS_AUTH_URL, OS_USERNAME, OS_PASSWORD,
OS_USER_DOMAIN_NAME, OS_PROJECT_NAME, OS_PROJECT_DOMAIN_NAME (and optionally
OS_REGION_NAME), or set openstack.createSecret=true.
{{- end }}
